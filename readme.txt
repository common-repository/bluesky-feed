Bluesky Feed WordPress plugin allows users to add feed to many social feed sites, mostly in Chinese.

== Description ==

<a href="http://bs.bluesky.cn">蓝天订阅按钮</a>是一个集合式中文RSS订阅集合按钮，用于侧栏Widget。可以让用户选择把RSS或者ATOM订阅到各种主流的中文FEED阅读服务上。。

== Installation ==

1. 将“bluesky_feed.zip”解压到`/wp-content/plugins/bluesky_feed` 目录下。
2. 在管理后台激活插件“BlueskyFeed订阅按钮 Widget”。
3. 在管理后台到：外观》小工具》拖放到侧栏
更多信息可以访问<a href="http://bs.bluesky.cn">蓝天收藏与分享主页</a>
